## What Lives Here

This is a **template repository** — users clone it to create their own private agent memory. Contributions here improve the starting experience for everyone.

**In scope for this repo:**
- README, QUICKSTART, and onboarding documentation
- The starter Meta agent template (`agent-meta/`)
- Repository structure and configuration

**For shared procedures, templates, and scripts:**
See [control-files/CONTRIBUTING.md](control-files/CONTRIBUTING.md) — the control-files submodule has its own contribution guidelines.

## How to Contribute

1. Fork the repository
2. Create a branch for your changes
3. Submit a pull request with a clear description

## Guidelines

- Keep the Meta agent template generic — no personal data or project-specific content
- Test that `git clone --recurse-submodules` works cleanly after your changes
- Ensure documentation is clear for first-time users
